Tipos de nós na árvore DOM


  
    

    
Explicações:

    
O código JavaScript faz o seguinte:

    
    
      
  • Itera sobre todos os nós filhos do elemento body e adiciona o protótipo do construtor de cada nó ao output.
    • 
      
  • Verifica se o elemento input é uma instância de várias classes e adiciona o resultado (true/false) ao output.
    • 
      
  • Adiciona o protótipo do construtor do documento ao output.
    • 
    

    
A saída será algo assim:

    
      [object HTMLHeadingElementPrototype]
      [object TextPrototype]
      [object HTMLInputElementPrototype]
      [object TextPrototype]
      [object HTMLPreElementPrototype]
      [object TextPrototype]
      true
      true
      true
      true
      true
      [object HTMLDocumentPrototype]
    

    
Explicação da saída:

    
    
      
  • [object HTMLHeadingElementPrototype]: O protótipo do elemento <h2>.
    • 
      
  • [object TextPrototype]: O protótipo de um nó de texto (espaço em branco ou texto entre elementos).
    • 
      
  • [object HTMLInputElementPrototype]: O protótipo do elemento <input>.
    • 
      
  • [object HTMLPreElementPrototype]: O protótipo do elemento <pre>.
    • 
      
  • true: O elemento input é uma instância de HTMLInputElement, HTMLElement, Element, Node e EventTarget.
    • 
      
  • [object HTMLDocumentPrototype]: O protótipo do objeto document.
    •
